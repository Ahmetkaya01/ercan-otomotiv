# Görseller

Site ve PDF sunumu bu klasördeki yerel görselleri kullanır.

## Firmanın gerçek fotoğraflarıyla değiştirme

Instagram: [@ercanalpay_otomotiv_insaat](https://www.instagram.com/ercanalpay_otomotiv_insaat/)

1. Instagram’dan araç, proje ve işletme fotoğraflarını indirin
2. Aşağıdaki dosya adlarıyla `assets/images/` içine kaydedin (üzerine yazın)
3. PDF’i yeniden oluşturun: `npm run sunum:pdf`

| Dosya | Kullanıldığı yer |
|-------|------------------|
| `hero-galeri.jpg` | Kapak + ana sayfa slayt |
| `hero-villa.jpg` … `hero-villa-3.jpg` | Ana sayfa arka plan |
| `world-otomotiv.jpg` | Otomotiv dünyası |
| `world-insaat.jpg` | İnşaat dünyası |
| `about.jpg` | Hakkımızda |
| `car-*.jpg` | Araç kartları (6 adet) |
| `proj-*.jpg` | İnşaat projeleri (4 adet) |
| `galeri-1.jpg` … `galeri-6.jpg` | Galeri |

Önerilen boyut: en az 1400px genişlik, JPG format.
